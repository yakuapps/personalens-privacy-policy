Privacy Policy for PersonaLens

Last updated: June 10, 2025
Effective date: June 10, 2025
Privacy-First Design: PersonaLens is built with privacy as a core principle. Your original chat files never leave your device, and we use advanced anonymization techniques to protect your identity during analysis.

1. Introduction

This Privacy Policy describes how PersonaLens ("we", "our", or "us") collects, uses, processes, and protects your information when you use our mobile application that analyzes WhatsApp conversations to provide personality insights and generate AI-powered visualizations.

App Information:

  • App Name: PersonaLens
  • Bundle ID: com.personalens.app
  • Platforms: iOS and Android
  • Developer: PersonaLens Team

2. Information We Collect

2.1 WhatsApp Chat Data

Privacy Protection: Your original WhatsApp chat files remain entirely on your device and are never uploaded or transmitted to our servers.

When you upload WhatsApp chat export files, we process:

  • Chat Messages: Text content from your conversations
  • Timestamps: When messages were sent (for pattern analysis)
  • Participant Information: Contact names (anonymized before any processing)
  • Message Metadata: Message length, response patterns, conversation flow
  • Media References: We note when media was shared but do not process the actual media content

2.2 Processed Analysis Data

For personality analysis, we send only optimized and anonymized versions of your chat data:

  • Anonymized Chat Content: Messages with real names replaced by generic identifiers (User1, User2)
  • Conversation Metrics: Message counts, response patterns, activity patterns
  • Processing Metadata: Token counts, sampling information, analysis parameters

2.3 App Usage Information

  • Feature Usage: Which features you access and how often
  • Analysis Preferences: Selected art styles, analysis types, settings
  • Report History: Generated reports and their status
  • App Performance: Crash reports, error logs, response times

2.4 Device Information

  • Device Identifiers: For subscription validation and app functionality
  • Operating System: iOS/Android version for compatibility
  • App Version: For support and feature availability
  • Network Information: For API communication (no browsing history)

2.5 Subscription Information

  • Purchase Data: Subscription status, purchase receipts, renewal dates
  • Billing Information: Handled entirely by Apple App Store or Google Play Store
  • Entitlements: Which features you have access to

4. Data Storage and Security

4.1 Local Storage (Your Device)

Local-First Approach: The majority of your data is stored locally on your device using secure, encrypted storage.
  • Original Chat Files: Never leave your device
  • Analysis Reports: Stored locally using MMKV encrypted storage
  • Generated Images: Cached locally on your device
  • App Settings: Stored securely on your device

4.2 Cloud Storage (Optional)

  • User Control: You can optionally enable cloud backup for your reports
  • What's Backed Up: Only generated reports and images (never original chat files)
  • Providers: Uses your device's native cloud storage (iCloud, Google Drive)
  • Encryption: All cloud-stored data is encrypted

4.3 Security Measures

  • Encryption in Transit: All network communications use HTTPS/TLS
  • Encryption at Rest: Local storage is encrypted
  • Data Anonymization: Real names are replaced before any external processing
  • Access Control: App data is sandboxed and inaccessible to other apps
  • Secure APIs: Authentication and rate limiting on all API endpoints

5. Third-Party Services

5.1 AWS (Analysis Processing)

  • Purpose: Processes anonymized chat data for personality analysis and image generation
  • Data Shared: Anonymized chat content, processing metadata
  • Data NOT Shared: Original files, real names, raw unprocessed content
  • Location: AWS US-East-1 region

5.2 RevenueCat (Subscription Management)

  • Purpose: Handles subscription purchases and validation
  • Data Shared: Device identifiers, purchase receipts, subscription status
  • Privacy: No personal conversation data is shared

5.3 Expo Services (Development Platform)

  • Purpose: App development framework and analytics
  • Data Shared: Standard app analytics, crash reports, performance metrics
  • Usage: App updates, performance monitoring, bug fixes

5.4 Apple App Store / Google Play Store

  • Purpose: App distribution and basic analytics
  • Data Shared: Standard platform analytics and download statistics
  • Control: Standard platform policies apply

6. Data Anonymization and Privacy Protection

Advanced Anonymization: We use sophisticated techniques to protect your identity while maintaining the analytical value of your conversations.

6.1 Participant Anonymization

  • Name Mapping: Real names are mapped to generic identifiers (User1, User2, etc.)
  • Consistency: The same person always gets the same anonymous identifier
  • Local Storage: Original name mappings are stored only on your device
  • Server Protection: Real names never reach our analysis servers

6.2 Content Optimization

  • Smart Sampling: Only relevant messages are selected for analysis
  • Token Limits: Content is optimized to fit within processing limits
  • Context Preservation: Conversation meaning is maintained while minimizing data
  • Quality Control: Parsing confidence scores ensure data integrity

7. Your Rights and Controls

7.1 Data Access and Portability

  • Report Export: You can export your analysis reports at any time
  • Image Export: Save generated artwork to your device gallery
  • Settings Backup: Export your app preferences and settings

7.2 Data Deletion

  • Individual Deletion: Delete specific reports, chat files, or images
  • Complete Wipe: Delete all PersonaLens data from your device
  • Automatic Cleanup: Temporary files are automatically removed
  • Cloud Deletion: Remove backed-up data from cloud storage

7.3 Processing Control

  • Selective Analysis: Choose which chat files to analyze
  • Feature Control: Enable or disable specific features
  • Cloud Sync: Control whether reports are backed up to cloud storage
  • Sharing Options: Control how and when reports can be shared

8. Data Retention

8.1 Local Data Retention

  • User Control: Data is stored until you choose to delete it
  • App Uninstall: All local data is removed when the app is uninstalled
  • Automatic Cleanup: Cache files and temporary data are automatically managed

8.2 Server Data Retention

  • Processing Only: Data sent to servers is used only for immediate processing
  • No Long-term Storage: We do not store your chat data on our servers
  • Temporary Processing: Data is deleted from servers after analysis completion

9. Children's Privacy

PersonaLens is not directed to children under 13 years of age. We do not knowingly collect personal information from children under 13. If you are a parent or guardian and believe your child has provided us with personal information, please contact us immediately.

10. International Data Transfers

When you use PersonaLens, your anonymized data may be processed on servers located in the United States (AWS US-East-1 region). We ensure appropriate safeguards are in place to protect your data during any international transfers.
